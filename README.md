## **Project 1: InnovateTech Website**

**Description:**  
A responsive multi-page website for a fictional company, demonstrating modern HTML5 semantic structure and CSS layout techniques.

**Features:**
- Homepage, About, and Contact pages.
- Semantic HTML5 elements: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`.
- Responsive layout with Flexbox and Grid.
- Styled forms with validation on the Contact page.
- External CSS organization with a `style.css` file.
- Accessible and SEO-friendly structure with proper `alt` text and labels.

---

## **Project 2: Interactive Dashboard Application**

**Description:**  
A responsive admin dashboard interface showcasing advanced CSS layout techniques.

**Features:**
- Fixed sidebar navigation with user profile section.
- Header bar with search, notifications, and profile dropdown.
- Statistics cards using CSS Grid.
- Data tables with zebra-striping, sticky headers, and summary rows.
- Quick Actions section with forms for adding products.
- Responsive design for mobile and tablet viewports.
- CSS custom properties, Flexbox, and Grid for layout management.
